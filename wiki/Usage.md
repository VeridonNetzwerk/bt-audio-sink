# Usage

## Main window

The main window is split into three areas:

1. **Header**
   App name and short description
2. **Paired Devices**
   List of detected Bluetooth audio devices
3. **Settings**
   Auto reconnect, startup behavior, start minimized

## Connecting a device

1. Find the device in the list.
2. Click **Connect**.
3. The status changes to **Connected**.

## Disconnecting a device

- Click **Disconnect** on the connected device.

## Status bar

At the bottom, the app shows the current connection status, for example:

- Ready
- Connecting...
- Connected to <device>
