# Bedienung

## Hauptfenster

Das Hauptfenster besteht aus vier Bereichen:

1. **Kopfbereich**
   - App-Name, Kurzbeschreibung
2. **Paired Devices**
   - Liste aller erkannten Bluetooth-Audio-Geraete
3. **Now Playing**
   - Aktueller Titel + Mediensteuerung
4. **Settings**
   - Auto-Reconnect, Autostart, Start minimiert

## Geraete verbinden

1. Geraet in der Liste suchen
2. **Connect** klicken
3. Status wechselt auf **Connected**

## Geraete trennen

- Beim verbundenen Geraet auf **Disconnect** klicken.

## Mediensteuerung

- **Zurueck**: vorheriger Track
- **Play/Pause**: Wiedergabe starten/anhalten
- **Weiter**: naechster Track

Die Steuerung laeuft ueber das Windows Media Session System (GSMTC).

## Statusleiste

Unten siehst du den aktuellen Verbindungsstatus, z. B.:

- Ready
- Connecting...
- Connected to <Geraet>
